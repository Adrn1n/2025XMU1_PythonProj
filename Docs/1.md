# 实验1 设计一个程序抽取百度的搜索页面信息

## 实验目的

为了实现具有实时联网搜索功能的 DeepSeek 大模型, 让它可以根据时事搜索到的结果进行回答, 需要先获得这些搜索结果的数据. 因此, 阶段 1, 即本实验要实现 Baidu 搜索结果的爬取.

## 实验要求

1. 网址：[https://www.baidu.com](https://www.baidu.com)
2. 需要抽取下列信息：
    网页标题、页面链接、网页摘要
